[ DISCORD MUSIC BOT ]

【 ОПИСАНИЕ 】
Музыкальный бот для Discord с поддержкой YouTube, очередью треков,
регулировкой громкости и русскими командами.

【 ТРЕБОВАНИЯ 】
- Python 3.8+
- Discord Bot Token
- FFmpeg (обязательно!)

【 УСТАНОВКА 】

1. Установите библиотеки:
   pip install discord.py yt-dlp

2. Установите FFmpeg в корневую папку проекта:
   
   Ваша-папка/
   ├── ffmpeg/
   │   └── bin/
   │       └── ffmpeg.exe
   └── bot.py

   ВНИМАНИЕ! Бот ищет FFmpeg по пути: ffmpeg/bin/ffmpeg.exe

【 НАСТРОЙКА ТОКЕНА 】

ОТКРОЙТЕ ФАЙЛ main.py, НАЙДИТЕ СТРОКУ 541:
ЗАМЕНИТЕ на свой токен:

   TOKEN = 'ВАШ_ТОКЕН_БОТА'

【 КАК ПОЛУЧИТЬ ТОКЕН 】
1. Discord Developer Portal -> New Application
2. Вкладка Bot -> Reset Token -> скопировать
3. Включить Message Content Intent (ОБЯЗАТЕЛЬНО!)
4. OAuth2 -> URL Generator -> bot + необходимые права

【 ЗАПУСК 】
python main.py

【 КОМАНДЫ 】

🎵 ВОСПРОИЗВЕДЕНИЕ:
!play / !играй <название>  - воспроизвести трек
!skip / !пропусти          - пропустить текущий трек
!next_track / !следующий   - переключить на следующий
!pause / !пауза            - пауза
!resume / !продолжи        - продолжить
!stop / !стоп              - остановить и очистить всё
!now / !сейчас             - информация о текущем треке

📋 ОЧЕРЕДЬ:
!queue / !очередь          - показать очередь
!clear / !очисти           - очистить очередь
!remove / !удали <номер>   - удалить трек из очереди
!shuffle / !перемешай      - перемешать очередь

⚙️ НАСТРОЙКИ:
!volume / !громкость [0-100] - изменить громкость
!loop / !повтор              - включить/выключить повтор
!leave / !уйди               - отключить бота

ℹ️ ПРОЧЕЕ:
!команды / !help           - показать все команды

【 ПРИМЕРЫ 】
!play Imagine Dragons Believer
!play https://youtube.com/watch?v=...
!volume 75
!очередь
!пропусти
!громкость 50
!повтор

【 РЕШЕНИЕ ПРОБЛЕМ 】

❌ FFmpeg не найден
   -> Проверьте, что папка ffmpeg лежит в КОРНЕВОЙ папке с ботом
   -> Путь должен быть: ./ffmpeg/bin/ffmpeg.exe

❌ Бот не видит команды
   -> Проверьте Message Content Intent в настройках бота

❌ Не воспроизводится звук
   -> Переустановите FFmpeg
   -> Проверьте интернет-соединение

❌ Ошибка импорта yt-dlp
   -> pip install yt-dlp --upgrade
